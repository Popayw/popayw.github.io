AdFind

AdFind是一款C++语言编写的域中信息查询工具，可以在域中任何一台主机上使用，在内网渗透使用率较高

语法格式如下：

其中，-b指定一个BaseDN作为查询的根，-f为LDAP过滤条件，attr list 为需要显示的属性。

执行以下命令：

查询 hack-my.com 域中的所欲computer对象，并过滤对象的name和operatingSystem属性

执行以下命令：

查询 hack-my.com 域中的所有 user 对象，并过滤对象的cn属性

常用查询命令

查询域中机器

  • 查询 hack-my.com 域的所有 computer 对象并显示所有属性

  • 查询 hack-my.com 域的所欲 computer 对象并过滤对象的 name 和 operatingSystem 属性

  • 查询指定主机的相关信息

  • 查询当前域中主机的数量

  • 查询当前域中被禁用的主机

  • 查询当前域中不需要密码的主机

  • 查询当前域中在线的计算机

查询域中用户

  • 查询 hack-my.com 域的所有 user 对象并过滤对象的cn属性

  • 查询当前登录的用户信息和 Token

  • 查询指定用户的相关信息

  • 查询当前域中用户的数量

  • 查询当前域中被禁用的用户

  • 查询域中密码永不过期的用户

  • 查询当前域中不需要密码的用户

查询域控制器

  • 查询当前域中所有域控制器(返回 FQDN 信息)

  • 查询当前域中所有只读域控制器

  • 查询当前域中所有可读写域控制器

其他查询

  • 查询所有的组策略对象并显示所有属性

  • 查询域信任关系

  • 查询 hack-my.com 域中具有高权限的 SPN
